# Kuku

Pure Python Desktop-Web Hybrid Business Application.

## Quick Start

```bash
uv run python main.py
```

Or use the launchers:

- **Windows**: `.\run_app.ps1`
- **Linux/Mac**: `./run_app.sh`

Then open [http://127.0.0.1:8000](http://127.0.0.1:8000).

## Stack

FastAPI · Jinja2 · HTMX · Bootstrap 5.3 · SQLite

## Testing

```bash
uv run pytest
```
